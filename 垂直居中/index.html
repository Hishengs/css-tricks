一、文字垂直居中

1. 单行文字垂直居中

具体方法就是将元素 line-height 设置成与其高度一致，此时文字就是居中的。但是缺点是不支持换行。

这一行文字是居中的，但是不能换行。

2. 多行文字垂直居中

display: table-cell; vertical-align: middle;

这一行文字是居中的，
而且可以换行 
    .display-table {
      height: 100px;
      display: table-cell;
      vertical-align: middle;
    }

二、元素垂直居中

1. 固定高度元素垂直居中

具有固定高度的元素实现垂直居中相对容易。

parent
child height
    .parent {
      height: 200px;
      position: relative;
    }
    .parent .child.height {
      height: 100px;
      background-color: #49a9ee;
      position: absolute;
      top: 50%;
      left: 0;
      right: 0;
      margin-top: -50px;
    }
